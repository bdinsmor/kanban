Material Menu

Menu with Icons

<md-menu>

Menus offer a list of options that are displayed when triggered. The position (before, after) and (above, below) is automatically decided, but can be overridden with e.g. x-position='before'.

Example:

<md-menu>
<button md-menu-item>
<md-icon>dialpad</md-icon>
</button>
</md-menu>
